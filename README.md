## ✨ 新功能 

与原始插件相比，本版本进行了一些功能更新：

- ### 增加简体中文支持
- ### 将小地图改为浮动窗口
    - 修复了可以透过小地图点击到底层白板的 bug
    - 小地图现在可以通过拖动标题栏随意移动
    - 小地图现在可以通过拖动边框更改整体大小
    -  " 位置 " 设置项改为仅设置一次预设位置，而非将小地图锁定在可选的预设位置之间
- ### 增加了侧边按钮，用于一键开关小地图
> [!CAUTION]
> 	😭 **小地图仅能在白板界面开关、更改配置，否则不生效 !** 😭
- ### 增加了全局透明度设置项
    - 更改了默认背景为无透明度背景，但仍能自定义
    - 全局透明度设置会连同背景一起设置
- ### 进行了简单美化
	- 更新了小地图的显示效果
	- 重新排列设置项
- ### <font color="#ff0000">删除了 " 劫持工具栏 " 功能</font>😥

## 📦 安装方法

### 通过 BRAT 插件安装
1. 在插件市场中安装 BRAT 插件
2. 打开 BRAT 插件设置
3. 选择 "Add beta plugin"
4. 在输入框中输入本仓库地址
5. 选择 "Add plugin" ，等待插件自动安装

---

## ⚙️ 使用方法

- 于白板界面打开命令面板，搜索 `Canvas minimap`，可进行**开关小地图**和**重载小地图**操作。
- 于白板界面在侧边栏点击插件图标，可切换显示/隐藏小地图。
- 在设置界面可以对小地图进行丰富的个性化设置
	- 颜色代码后可输入两位数字设置透明度
